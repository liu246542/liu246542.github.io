[一周总结] 19-07-03 ～ 短锄栽花

发表于 2019-07-03 | 分类于 管中窥豹 | 评论数：
短锄栽花，长诗佐酒，
诗剑年年总忆卿；
天山上，看龙蛇笔走，
墨泼南溟。
          -- 梁羽生
          「白发魔女传」

六月是个一言难尽的月份，但尤其适合念诗。因此这次我就没有用图片作为封面，而是截取了一段词。说起来也好笑，这段词我是高中时代背的，但由于那个时代盗版书籍猖獗，把 短锄栽花 写成了 短栽花，让我一时不解。直到大学在图书馆里翻到正版书籍，才发现原来是漏掉了一个字，顿时多年的困惑得以化解。这也算是盗版受害者的真实案例吧。

1. 《唐诗百话》

“龙文虎脊皆君驭，历块过都见尔曹”

一提到诗，我就不禁想起 “苟”～。 打住，今天我们不养“苟”，只谈论唐诗。为了提高大家的知识水平，我个人超级推荐读一下 施蛰存 老先生的 《唐诗百话》。推荐这本书呢，最主要是因为作者对唐诗有着自己独到的见解与观点，而不是高中语文那种标准答案式的套词。

评判一本书的好坏，最暴力的方式就是看这本书的作者是否在传达自己的主张，并利用一系列的证据来支撑起自己的主张。如果通过阅读一本书，可以很明显地感受到这种主观性，那么这本书大抵就是值得一读的。

比较有印象的就是先生对 杜甫 的《戏为六绝句》的评论，其中有四句已被千古传诵：

王杨卢骆当时体，
轻薄为文哂未休。
尔曹身与名俱灭，
不废江河万古流。

但是对这四句的解释有着许多截然不同的声音。于是乎，顾老先生就站了出来，把这些胡乱评注的人都批判了一番。摘录一下原文：

这样一首清清楚楚的诗，几十年来，我从来没有想到它可以有不同的诠释。但是，读了郭老所辑录的历代注家的评论，却打开了眼界。
…
一句文从字顺的七言诗，被列位杜诗学者讲得支离破碎，又通过郭老的分析评断，做出结论，几乎成为标准解说。于是今天在各大学中文系讲授文艺理论或杜诗的教师，都在这样讲、这样教、这样诠释。岂非怪事！

像这样的例子还有很多，大家可以从书里找到顾老的解答。我要是高中看到这本书，语文也就兴许能考及格了呢（笑。

2. web-based notebook

“B/S 主导一切！”

自从 Jupyter 出现之后，许多基于网页开发的交互式编程笔记本也变得多了起来。因为 Jupyter/Jupyterlab 本身也是使用浏览器作为客户端的，因此如果我们将一台强大的电脑作为后端，那么平时只要带着自己的笔记本甚至平板无忧无虑地进行简单的数据分析等任务了。理论上说只需要有安装了浏览器的终端即可，B/S 架构的优势就体现了出来。下面我就简单列出几个还不错的web端编程笔记本。

2-1. CODEPEN

严格意义上说，codepen 也算不上一个笔记本，但是这里提到它是因为 codepen 是写前端的好帮手。如果是想快速地构建一个网页框架，那么 codepen 绝对是不二之选。codepen 提供了 html、css、JavaScript 三块编辑区域，而且有实时预览的功能。类似于 github，codepen 也有大量的网页实例，如果直接在别人的基础上构建，只需要点击 fork 即可。

2-2. Observable

Observable 我以前也提过，这个项目是由 d3.js 的创始人发起的，目前还在不断完善中。主要面向 JavaScript 编程，可以看成支持 JavaScript 的 Jupyter。

由于 JavaScript 的灵活性，用户可以构建许多有意思的小项目，当然，不嫌麻烦的话，画图也是很好看的。

官方有个简洁的 Introduction 教程， 如果有用过 node.js 的经验，那么上手起来就会比较快。

2-3. Colaboratory

Colaboratory 是完全由 谷歌公司 开发的 类Jupyter 笔记本，谷歌还是财大气粗，用户可以使用谷歌云盘关联自己的文件。当然，因为是谷歌嘛，用不了我也没办法。

2-4. Iodide

Iodide 是一个很神奇的工具，用户可以混合使用 Python 和 JavaScript 语言，简直是双倍的快乐！

2-5. binder

binder 相比前面几个可能不是那么“动态”，它是可以把 github 仓库中的 Jupyter文件 实时执行，这在做 presentation 的时候，可能会有一些帮助。

4. 《刀说异数》上篇完结

“世事如棋，乾坤莫测，笑尽英雄啊！”


其实我入布袋戏的时间比较晚，要补旧剧是不可能的事了。《刀说异数》作为老剧翻拍，有着更为工致的木偶和更华丽的特效，极大地满足了我的需求。而且闽南语听着很带感啊，就是喜欢这种装逼中带着中二气息的感觉。

为什么推荐布袋戏呢，有这么几点原因：

  1. 布袋戏很好地发扬了中华文化，包括人物的取名、服饰、诗号以及音乐等等；
  2. 布袋戏在人物塑造方面很不错，江湖也不是非黑即白，所谓的正派人士往往也是沾满鲜血，反派人物又有着可怜一面；
  3. 悲剧的基调 （总的来说，拥有悲剧内核的故事才是我的菜）
  4. 因为小众，所以可以装逼 （我从未见过有如此厚颜无耻之人

5. E3 快乐周

任天堂就是世界的主宰！(爆音)

要说到六月，就不得不提到 E3游戏展 肥宅快乐周。今年索尼没有参展，正应了粉丝口中的“闭关索国”、“离群索居”。不管怎么样，少了大法，E3也不会减一秒。今年老任也没有让大家失望，一举拿出了 路易基鬼屋、火焰纹章、动物之森 这样的新作，之前巫师3登录ns平台的消息也得到证实。最惊喜的莫过于结尾处的塞尔打传说续作，虽然连个饼的壳子都没有看到，但至少老任表示“在做了”。还要什么，大喊“老任nb”就行了。

其实让我关注的不是这件事，而是国内某个游戏媒体在做E3直播的时候，接二连三地认不出是什么游戏，更搞笑的是在预告片播放的同时，玩弄着以前调侃的老梗。这也就算了，关键是事后的声明把问题归咎为年龄问题，转移问题的重点。公关做到这样，我只能说：

因此，如果要看跟游戏相关内容还是关注一些比较专业的up主，比如 机核、游戏时光（vgtime）。另外安利一下 红酒汤姆一世，高级膜法师加上真正的任粉，晚上睡觉之前听一听绝对会睡不着。

好久没有写这些总结，平时脑子中都是些光怪陆离的想法，有时候也是希望跟大家分享一下。文字作为我能用到的最方便的载体，可以记录我在某些阶段的一些观念，不一定是对的，只是权当流水帐了。可能对我来说，传递想法比传递基因更重要吧。（所以我还没有女朋友。。。）
