[杂谈] Arch 上好用的软件

发表于 2018-12-30 | 更新于 2020-11-03 | 分类于 折腾小记 | 评论数：

Arch Linux 作为 Linux 众多发行版中的一朵奇葩，拥有着一个庞大的软件库，有篇文章 ➢什么Linux发行版软件最多 就比较了debian系，red-hat系和Arch系的软件库。很明显可以看到Arch在软件库方面的强大之处。所以今天的标题中强调的是Arch，虽然其他版本的Linux也可以使用这些软件，但在Arch上面，一句命令就可以安装好。

在介绍软件之前，先由我简单解释下Arch上软件库庞大的原因，然后再介绍下Arch上安装软件的一些基本命令。

0. Aur 和 pacman

“等等， pacman 不是吃豆人那个单词吗？”

Aur，即 Arch user repository 的缩写，翻译过来就是arch用户的软件仓库。之所以Arch上的软件那么丰富，其中一个主要原因就是Arch上使用的仓库来源于用户的创建。Arch官方会定期从中挑选一些软件包到community仓库。正是因为这种由用户主导的方式，Arch上的软件库才会一枝独秀。

安装官方的软件，需要用到的基本命令就是 pacman，如果安装Aur中的软件包(囊括了官方的仓库)，我比较推荐使用 yay。二者使用方法一致，唯一的区别就是 yay 不需要在命令前加上 sudo，一般是等后面有提示的时候再输入密码。

官方的wiki提供了一个比较全的列表，我接下来介绍的一些软件纯属个人平时觉得比较好用的，当然更全面的还是参考这个列表➢List of applications。

1. FreeOffice

一个office的替代品，➢官方地址

Linux 上有许多office的替代品，比如wps，libreoffice等。但是我比较满意的还是这款叫做 freeoffice的软件。个人觉得编辑的工作还是交给 Windows，在Linux上就进行简单的查看工作就行了。相比于其它同类型的软件，freeoffice的兼容性还是比较不错的。

值得注意的是，第一次安装后打开需要输入激活密钥，而密钥需要在官网输入自己的邮箱获取。

2. Typora

最好用的markdown编辑器

Typora 是一款基于 markdown 语法的文字编辑器。不同于传统的双栏显示， Typora 真正意义上做到了所见即所得。

Typora 的功能十分强大，对数学公式的支持也很好，而且 ➢官网上 有多种主题提供下载，有能力的也可以自己编辑 css文件，创造属于自己的样式。

这里补充一句，Typora支持全平台，所以说它是最好用的 markdown 编辑器也不为过。

3. Telegram

我跟微信boy聊不来的，走了

如果让我推荐一款聊天软件，那我首先推荐的就是telegram，中文翻译为电报，圈内简称为tg(不要跟那个tg混淆)。

我使用tg的时间并不长，但自从用了之后就欲罢不能。相比于微信来说，它有两个最大的优势，一是全平台，几乎所有的平台都有桌面客户端；二是消息同步，不同于微信蹩脚的消息同步机制，tg几乎能看到所有的历史消息(甚至在没有加群之前，群内的消息也是可以看到的)。tg有一个很强大的后盾，即机器人(bot)的功能。

tg上的机器人主要是推送消息用的，有点像微信的公众号的作用。由于tg的开放性，用户可以获取到许多未经“审查”的信息。另外，用户也可以自己构造机器人，只要有点编程的知识，这个也不是很困难的事。

tg自由开放的特点注定它不能在中国使用的。但官方给出了MTproxy协议，让我等屁民可以一窥tg的全貌。据说这个协议对外进行了悬赏，以证明其拥有足够的安全性。源码地址见gayhub，对于Debian系用户，按照过程一步步搭建应该没问题，但对于Red Hat系用户，我之前有遇到g++版本太低的问题，可以参考➢配置MTProxy来访问Telegram解决。

另外，tg上的贴纸真的很有趣。

4. GeoGebra

都9102年了，你还在用几何画板？

记得上初中的时候，我们用的教科书上还有介绍几何画板这个工具，当时还颇感震撼。GeoGebra就是很类似的一款软件。当然，由于几何画板收费的缘故，我一直没有机会使用，但GeoGebra还是非常容易上手的，而且也支持全平台。平时如果想画一些简单的函数图像，而且不想靠编程实现的话，可以考虑一下。

知乎上的 马同学 就凭借着这款软件画得一手好图，大家可以参考参考。

5. Tor browser

上暗网？下它就对了

前些日子网传12306网站出现用户信息泄漏，这些数据最初是挂在暗网上贩卖的。大家肯定比较好奇怎么才能打开暗网网站。这里需要用到的就是Tor浏览器，中文一般称为洋葱浏览器。在中国使用Tor还需要挂代理，而且一般打开网页的速度会很慢。

除了访问暗网，我们也可以使用Tor浏览器来保护我们个人的隐私。

顺带贴一下中文交易论坛的地址 ➢暗网市场交易站 ，不要干坏事哟。

6. Vivaldi 浏览器

“极客浏览器”

如今浏览器的市场俨然成为了Chrome的天下，我个人也比较推荐使用Chrome或Chromium浏览器作为个人电脑的默认浏览器。今天在这里推荐的Vivaldi浏览器可能大家没怎么听过，虽然它也是基于Chrome的内核，但有着丰富的可定制内容，其最强大的就属于侧边栏的加入。

侧边栏可以放置任意的网页面板，像这样，我们可以在浏览网页的同时，听着自己喜欢的音乐。或者也可以打开一个Bing词典面板，这样碰到一些生词可以比较方便地进行查询。

另外一个比较实用的就是笔记功能，用户可以随时随地记下自己的笔记，而且可以任意地进行截图。笔记的语法基于markdown，因此上手基本零难度。

其它的比如自带截图，自带鼠标手势这些我就不一一赘述了，毕竟在很多国产浏览器中这些功能属于“标配”。更多自定义的东西大家可以自己去探索，总体来说，这款浏览器还是非常良心的。

另外，老毛子的 Yandex 浏览器也挺好用的，而且是全平台的。

7. Peek

“Linux上的GifCam”

Windows上有一款神器 ➢GifCam，可以录制屏幕为GIF动图格式。在Linux上也有着类似的一款工具 Peek。界面非常简洁，大家直接上手使用即可。

顺带吐槽一下Ubuntu，竟然没有在软件库中加入Peek。

8. QComicBook

漫画爱好者的福音

本人很喜欢看漫画，以前用Windows的时候还购买了Cover-漫画阅读器，后来转战Linux，才发现有免费的软件，也就是 QComicBook，基本能够满足我的要求。

开源软件就是好，感觉自己又亏了。

9. 网盘

8秒视频了解一下

提到网盘，大家首先会想到的是 百度网盘，Onedrive，Dropbox等一些比较主流的网盘。现在国内的网盘并不好做，说不定哪一天发个通知就溜了。我也尝试着搭建自己的网盘，比如像seafile和nextcloud。用了一段时间的 seafile，发现自己最需要的就只有两个功能：

  • 类似git的 版本控制
  • 直接挂载成 本地磁盘

我对容量要求不高，因此有两个网盘比较适合我，一是国外的 ➢mega，另外一个就是国内的 ➢坚果云。
