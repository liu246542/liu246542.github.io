[每月总结]19-03-30

发表于 2019-03-30 | 更新于 2020-12-03 | 分类于 管中窥豹 | 评论数：

这个月主要忙于赶毕业论文，很多东西都没来得及打理，立下的诸多flags也没有完成，外加对游戏失去了兴趣，想想活着真痛苦啊。

“That which does not kill us, makes us stronger.” – Friedrich Nietzsche

不管怎么样，总结还是要写滴。本来是打算周更的博客，硬是被我写成了月更。好吧，那就看看这个月上个月的沙雕总结。

1. 微小的工作

“很惭愧，只做了微小的工作”

这个月除了写论文也没干别的，硬是要说的话，大概有这三件事：

  1. 新建了一个笔记系统，使用的是 MkDocs，主要是对之前的学习做一些总结，初衷还是为了提高自己的知识水平。地址 ➢Lius Notes；
  2. 把博客界面进行了小改，所有的源文件也同步到了 Github 上，这里就不贴地址了，背景用的是本科时候自己仿的一段 CSS 代码，当时沉迷于一本叫做 CSS揭秘 的书籍，用的就是其中的技巧；
  3. 把以前荒废的直播间重开了起来，重点是这次加入了弹幕，直播地址 ➢Bilibili，用到的工具有 danmaku2ass 和 ffmpeg。

如果说还做了点什么事，就是把 鬼泣5 过了一周目，这对我来说是很有意义的。因为期待了好几年了，这次 冷饭之王 动作天尊 卡普空 也没有让玩家失望。通完一周目之后开始对游戏很消极，只有塞尔达传说能给我带来一点乐趣，但这些都是次要的，这个月主要就上面这三件事，很惭愧，就做了点微小的工作。

2. 先锋文学

本质上说，我是一个离经叛道的人

在完成论文和速通了游戏之后，整个人仿佛进入了“贤者模式”，对任何事物都提不起兴趣。在提不起劲的日子里，人就比较容易成为一个哲学家，脑子里会充斥着光怪陆离的想法。我回想起高中时代的日子，那时候我偏爱余华的短篇小说，像 十八岁出门远行，古典爱情 这类。这是我第一次接触先锋文学，瞬间就迷上了这种荒诞不羁，直接露骨的叙述方式。
本质上说，我是一个离经叛道的人，先锋文学在某种意义上助长了我的离经叛道。就个人而言，我比较喜欢 古典爱情 这篇。主要讲述的是书生与小姐的故事，外加一个丫鬟刚好是西厢记的标准配置。同时夹杂着肢解、食人肉这些桥段，最后当我以为是大团圆的结局时，又笔锋一转，打破传统的故事模式。总体来说，余华在写起故事来有如行云流水一般，一个荒诞的故事总能在他的笔下显得那么自然。印象里还有一篇名为 一九八六年，围绕着中国古代的五刑描述了70年代那场运动带来的创伤。
相比于之前“哭哭啼啼”式的伤痕文学，先锋派就像一把流星锤，直击对手的软肋。

我记得自己在外面高高兴兴地玩了半天，然后我回家了，在窗外看到父亲正在屋内整理一个红色的背包，我扑在窗口问：“爸爸，你要出门？”
父亲转过身来温和地说：“不，是让你出门。”
“让我出门？”
“是的，你已经十八了，你应该去认识一下外面的世界了。”
后来我就背起了那个漂亮的红背包，父亲在我脑后拍了一下，就像在马屁股上拍了一下。于是我欢快地冲出了家门，像一匹兴高采烈的马一样欢快地奔跑了起来。
————《十八岁出门远行》

3. 世嘉

万年老二

自从接触了如龙系列，开始对世嘉（SEGA）这个公司产生了兴趣。以前我对世嘉没有太深刻的印象，只知道他们出过一个名为 MD 的主机。通过翻看前几次的主机大战，我开始同情世嘉惨痛的经历。像世嘉这样的 “耿直boy” 要在激烈的市场竞争中存活下来实在太难了，尤其遇上了任天堂和索尼这种对手。

故事可以追溯到上世纪80年代，自从雅达利崩溃之后，任天堂抓住时机一跃成为家用机霸主。在任天堂一家独大的日子里，早期的第三方游戏厂商只能向其俯首称臣，屈服在老任的霸王条款之下。这些第三方厂商中有我们耳熟能详的 CAPCOM （卡普空，卡表，冷饭王）、KONAMI （科乐美）、NAMCO（南梦宫）、Square （史克威尔）等等，只是除了世嘉。

世嘉当时推出了自己的第一款家用游戏机 SG-1000 来对抗老任的 FC 红白机。SG-1000 在硬件设计上远不及红白机 （老任的控制器设计得很完美），而且由于缺少游戏内容的支持（游戏开发商基本都被任天堂垄断了），最终以惨败收场。

在任天堂推出了掌机 GameBoy 系列之后，“头铁”的世嘉以不服输的气势推出了自己的掌机 Game Gear，并顺带嘲讽了GameBoy的黑白无背光屏幕。但 Game Gear因为糟糕的续航和缺少足够的第三方游戏输给了 GameBoy。

随着游戏主机第五世代的到来，世嘉也迎来了新的对手——索尼。索尼是上世纪90年代中期开始进军家用机行业的，今天已经是游戏主机三巨头之一了。所以结局很明显，世嘉以 SEGA Saturn 和 Dreamcast 来分别对抗索尼的 PS1 和 PS2，最终均以彻底失败告终。与任天堂不同，索尼选择了乘胜追击，一举将世嘉彻底打垮。从此世嘉只专注于游戏软件的开发，为其它平台提供自己的游戏。

在这几次的游戏主机大战中，世嘉始终选择以正面对抗对手，当自己的硬件做的不够好的时候，就努力提高硬件性能（至少世嘉后面出的几台主机在性能上不比当时其它主机弱的）；当软件（游戏）不及的时候，努力研发游戏，也拥有了像 索尼克 和 莎木 这样的 IP。世嘉不是那种不会反思的公司，他们已经在不断地改进，只是遇到了两个“神”级的对手。在欧美市场，还是有不少的世嘉粉丝。很遗憾的是，我没有机会接触这个时代，因此也很难体会当 莎木3 公布的时候他们激动的心情。对他们来说，世嘉已经不止是一家公司，更象征着一个符号，一种始终坚持梦想的“硬派”精神。世嘉，虽败犹荣！

“我们可以一次一次去撞南墙，我们不能一个一个失去理想。”

4-2. Meme

Meme 是什么？Meme可以是一个表情包（熊猫头），一个耳熟能详的句子（今年下半年），一段洗脑的音乐（啊啊啊，啊啊啊，西湖美景，三月天呐）…
Meme 对应的汉语翻译为 模因，《自私的基因》一书中将其定义为 “一种文化传播单位或模仿单位”。

模因更像汉语中“梗”这个词的概念，很容易进行复制传播，比如“六学”狂欢，大秦话。
正因为这种易于传播的性质，不好的Meme会像病毒一样侵蚀人们的思想，可能有点危言耸听。
但想一想宗教的例子，宗教中的信仰也可以看作一种模因，多少人死磕着迂腐的信条，做出了多少不符合常人的行为。
